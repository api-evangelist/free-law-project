# Free Law Project (free-law-project)
Free Law Project is a non-profit organization that increases access to justice and transparency in the legal system through technology and open data. Its CourtListener REST API provides programmatic access to case law, PACER data, the RECAP archive, oral arguments, judges, financial disclosures, citations, citation networks, alerts, and tags.

## APIs

### Free Law Project API
The CourtListener REST API v4 provides access to opinions, dockets, RECAP documents, oral arguments, judges, financial disclosures, citation lookups, search, alerts, and tags. Authentication uses HTTP token (`Authorization: Token <token>`); rate limits are 5,000 queries/hour for authenticated users.

**Human URL:** [https://free.law/](https://free.law/)

**Base URL:** `https://www.courtlistener.com/api/rest/v4`
